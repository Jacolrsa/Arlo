# Arlo

Arlo is a Home Assistant integration for the MeshCore network.

Its goal is to provide intelligent MeshCore automation, chat commands, games, and utilities for both private and public MeshCore channels.

---

## Current Status

🚧 Under active development.

---

## Planned Features

- MeshCore Monday
- Leaderboards
- Weekly Streaks
- Achievements
- Weather
- Network Information
- AI Assistant
- Plugin System
